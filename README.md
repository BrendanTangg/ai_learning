# ai_learning
This is a compilation of AI agents I made while learning about AI agents. You need to install these packages to run the code.

python3 -m pip install crewai==0.28.8 crewai_tools==0.1.6 langchain_community==0.0.29

This includes a research+write article ai agent, a customer support automation ai agent, a customer outreach campaign ai agent, an ai agent that automates event planning, a financial trading ai agent, and an ai agent that can help improve a resume.
